# OpenCROW Utility Toolbox

Use this reference when a workflow is bottlenecked by shell manipulation rather than challenge-specific logic.

## Tool selection

- `jq`: JSON extraction and transformation in shell pipelines.
- `yq`: YAML and structured-data extraction.
- `xxd`: hex dumps and reverse conversion.
- `tmux` / `screen`: persistent multi-shell workflows.
- `rg`: fast recursive search in challenge trees or code.
- `fzf`: interactive fuzzy selection for files, history, or candidate values.
