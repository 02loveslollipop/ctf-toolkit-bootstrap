# OpenCROW Stego Toolbox

Use this reference when the challenge file likely hides data inside an image or audio carrier.

## Tool selection

- `zsteg`: PNG and BMP channel/bit-plane analysis with many built-in payload probes.
- `steghide`: embed or extract payloads from supported image/audio formats, often with a passphrase.

## Full-profile manual tools

- `StegSolve`: GUI bit-plane and color-channel inspection.
- `OpenStego`: general-purpose steganography tool with both GUI and CLI usage.
