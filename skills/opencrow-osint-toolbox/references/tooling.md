# OpenCROW OSINT Toolbox

Use this reference when the challenge expects reconnaissance from public data sources rather than direct interaction with the target.

## Tool selection

- `sherlock`: enumerate whether a username exists across many public sites.
- `waybackpy`: inspect archived snapshots and timestamped versions of websites.
- `shodan`: query internet-exposed services and banners through the Shodan API.

## Full-profile manual tools

- `theHarvester`: tracked by the installer as a manual full-profile step for broader email/host/subdomain harvesting.
