# OpenCROW Forensics Toolbox

Use this reference when the challenge artifact is a memory dump, disk blob, or file whose metadata may hide the answer.

## Tool selection

- `volatility3`: memory forensics, process/module listings, registry-style extraction, and malware triage from RAM dumps.
- `exiftool`: metadata extraction from images, documents, media, archives, and container formats.
- `foremost`: file carving by header/footer when the challenge blob contains recoverable embedded content.

## Full-profile manual tools

- `Autopsy`: tracked by the installer as a manual full-profile step for GUI-first disk and timeline workflows.
